hello and welcome!

this project is a go backend, svelte frontend web application with a postgres db and a caddy reverse proxy.

this app is basically an empty template. the backend has a health check endpoint and the frontend is empty. it's your empty canvas.

before running the app, run:

`cp .env.template .env` to initialize an empty .env file

then, run:

`docker compose up --build`

this will set up all the components of your development environment. okay, have fun!
